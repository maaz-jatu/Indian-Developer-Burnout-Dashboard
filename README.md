# Indian Developer Burnout & AI Anxiety Dashboard

## Project Overview

This project analyzes burnout, stress levels, AI anxiety, and job-switch intent among 5,000 developers using Microsoft Excel.

The goal was to transform raw workforce data into meaningful insights through data cleaning, KPI design, pivot analysis, and dashboard storytelling.

---

## Key Insights

* Startup and freelance developers showed the highest burnout levels.
* Longer weekly work hours strongly correlated with burnout.
* Approximately 31% of developers indicated job-switch intent.
* Medium burnout risk was the most common category overall.
